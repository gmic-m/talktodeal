01 / Overview

A reference platform engineered for voice in the field, not on the desk.

The MIC 05 is a production-ready voice capture module designed for OEMs who need studio-grade audio in non-studio environments — vehicles, factory floors, retail counters, vibration, wind, gloves.

Its dual-array MEMS configuration runs on-device beamforming and adaptive noise reduction through a dedicated DSP pipeline, so usable audio reaches your stack without cloud preprocessing. The 1200 mAh cell delivers 20 hours of continuous capture, and the module operates standalone over Wi-Fi 6 — no companion device on the customer's belt.

GIMC manufactures MIC 05 at our Shenzhen facility with full ODM scope: enclosure, branding, firmware, BOM modifications, and target-market certifications.

04 / API & Integration

Four wire-level integration surfaces.

MIC 05 ships with public, documented integration points. You own the audio. You own the data plane. We supply the silicon and the SDK.

REST · v1
Control plane.
HTTPS endpoints for provisioning, fleet management, device telemetry, and OTA orchestration. OAuth 2.0 client credentials. Stable v1 contract at api.gimc.io/v1 with 12-month deprecation policy.
WebSocket
Real-time audio & events.
Bidirectional channel for streaming PCM, VAD events, and low-latency control commands (LED, mode, sample rate). ≤ 80 ms median round-trip on Wi-Fi 6; back-pressure aware with framed chunking.
Webhook
Push notifications.
HMAC-SHA256 signed payloads for button presses, charge state, OTA result, low-battery, fault codes. Exponential backoff with 24 hr retention; configurable per event type.
SDK
First-party libraries.
TypeScript / Node.js, Python 3.10+, C (embedded host), Swift, Kotlin. Same surface across runtimes. Source-available on partner GitHub org; semver-locked releases.
device.connect.js · TypeScript 
import { MIC05 } from '@gimc/mic05';

const device = new MIC05({
  apiKey: process.env.GIMC_API_KEY,
  serial:  'M05-A4F92K-0117'
});

// Connect over LAN; falls back to BLE if Wi-Fi unavailable
await device.connect();

await device.audio.configure({
  sampleRate:     96_000,
  bitDepth:       24,
  beamforming:    'auto',   // 'auto' | 'fixed' | 'off'
  noiseReduction: 'high'     // 'off' | 'low' | 'high'
});

device.on('button.long_press', async () => {
  await device.led.set('breathing', '#1A6B5A');
  const session = await device.session.start();
  console.log('capture started', session.id);
});

device.audio.on('chunk', (chunk) => {
  // chunk = { ts, pcm, channel, rms, vad }
  if (chunk.vad) uploadPCM(chunk.pcm);
});

device.on('battery.low', async ({ percent }) => {
  console.warn(`battery at ${percent}% — switching to eco`);
  await device.power.mode('eco');
});
